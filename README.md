# About
大学时写的一些数据结构和算法集合，大部分用c语言实现。

```
├─bezier
├─biTree    //二叉树
├─biTree-NoRecursion-Threah     //非递归实现二叉树遍历、线索化二叉树
├─kmp       //kmp
├─merge-K-SortList      //合并k个有序链表
├─pathFinding       //寻路算法
├─queueLink     //队列
├─removeNth-FromEnd     //去除链表倒数第n个数
├─sequenceTable-LinkedList      //链表
├─sort
│  ├─binaryInsertSort       //二分插入排序
│  ├─bubbleSort     //冒泡排序
│  ├─heapSort       //堆排序
│  ├─indirectInsertSort     //直接插入排序
│  ├─indirectSelectSort     //选择排序
│  ├─mergeSort      //并归排序
│  ├─quickSort      //快速排序
│  └─shellSort      //希尔排序
└─stackLink     //栈
```
